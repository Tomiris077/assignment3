# Assignment 3 - Algorithms Analysis

## Algorithms Used

1. Bubble Sort
   - Time Complexity: O(n^2)
   - Simple but slow for large data

2. Merge Sort
   - Time Complexity: O(n log n)
   - Efficient and stable sorting algorithm

3. Binary Search
   - Time Complexity: O(log n)
   - Requires sorted array

## Program Structure

- Sorter.java: contains sorting algorithms
- Searcher.java: contains binary search
- Experiment.java: generates arrays and measures time
- Main.java: runs experiments

## Experiment

Arrays of size:
- 10
- 100
- 1000

Results show:
- Bubble Sort becomes very slow as size increases
- Merge Sort performs efficiently
- Binary Search is very fast

## Conclusion

Efficient algorithms like Merge Sort and Binary Search are much better for large datasets compared to simple algorithms like Bubble Sort.
